Complete feature development workflow from planning to implementation.

## Process

1. **Use architect agent** to analyze requirements and create implementation plan
2. **Review plan.md** with user for approval
3. **Use test-engineer agent** to create failing tests for the feature
4. **Use builder agent** to implement the feature following the plan
5. **Use reviewer agent** to validate code quality before commit

## What I need from you

- Feature description or requirements
- Any specific constraints or preferences
- Which parts of the codebase might be affected

## What you'll get

- Detailed implementation plan (plan.md)
- Complete test suite (passing)
- Working implementation
- Code review report (review.md)
- Commit-ready code

## Example usage

/project:feature "Add user profile page with avatar upload and bio editing"
